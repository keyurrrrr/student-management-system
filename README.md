# 🌦️ Weather App

## 📌 Overview

The Weather App is a desktop application developed in Python using Tkinter. It fetches real-time weather information from the OpenWeatherMap API and displays temperature, humidity, and weather conditions for any city.

## 🚀 Features

- Search Weather by City
- Real-Time Temperature
- Humidity Information
- Weather Description
- User-Friendly GUI
- Error Handling

## 🛠️ Technologies Used

- Python
- Tkinter
- Requests Library
- OpenWeatherMap API

## 📂 Project Structure

```
WeatherApp/
│
├── main.py
├── README.md
```

## ▶️ How to Run

Install the required package:

```
pip install requests
```

Run:

```
python main.py
```

## 📸 Output

Displays:

- City Name
- Country
- Temperature
- Humidity
- Weather Condition
